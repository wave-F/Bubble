BGM/SFX preview pack for casual match-3/sort game

Recommended mapping:
- Main menu BGM: main_menu_heavenly_loop.ogg
- In-level BGM: level_underwater_theme.mp3
- Win result SFX (short): sfx_win_pleasing_bell.wav
- Lose result SFX (short): sfx_fail_beep.wav

Optional long result track:
- result_win_soft_carefree.mp3
- result_lose_project_utopia.ogg
